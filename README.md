# Data Science & Machine Learning

This repository contains advanced solutions and research in the fields of Data Science, Deep Learning, and Artificial Intelligence, focusing on building end-to-end pipelines from data preparation to generative AI models.

## 📌 Description
The projects and exercises in this repository are aimed at solving complex analytical problems using modern machine learning frameworks. The tasks cover a wide range of scenarios, including predictive modeling, natural language processing, and the implementation of state-of-the-art generative architectures like RAG.

## 🧠 Key Concepts and Techniques

Within the repository, the following topics and methodologies are covered:

### 🔹 Deep Learning & Generative AI
* **Neural Networks:** Building and training models using **PyTorch** (CNN, LSTM architectures).
* **Transformers:** Implementation and understanding of attention-based models.
* **Generative AI:** Advanced retrieval techniques using **RAG (Retrieval-Augmented Generation)** and **Agentic RAG**.

### 🔹 Machine Learning & Predictive Modeling
* **Supervised Learning:** Classification (e.g., Churn Prediction) and Regression tasks.
* **Ensemble Methods:** Boosting and Bagging techniques for improved model performance.
* **Time Series:** Forecasting and analyzing sequential data patterns.
* **Unsupervised Learning:** Clustering (K-Means) and Dimensionality Reduction (Truncated SVD).

### 🔹 Natural Language Processing (NLP)
* Text Vectorization and Feature Extraction using **TF-IDF**.
* Working with sequence data and advanced NLP architectures.

### 🔹 Data Engineering & Preprocessing
* **Data Imputation:** Handling missing data with **KNNImputer**.
* **Feature Engineering:** Scaling (StandardScaler), Encoding (OneHot, Label, Ordinal), and Dimensionality Reduction.
* **Exploratory Data Analysis (EDA):** Deep analysis and visualization of statistical patterns using **Seaborn** and **Matplotlib**.

## 🎯 Repository Goal
* Practical application of **AI/ML** concepts through concrete implementations.
* Understanding the mechanics of **Generative AI** and **RAG** systems.
* Developing scalable and efficient data pipelines for real-world datasets.
* Mastering model evaluation, performance tuning, and metrics.

## 🛠 Technologies
* **Language:** Python
* **Deep Learning:** PyTorch
* **Machine Learning:** Scikit-learn
* **Data Handling:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook
